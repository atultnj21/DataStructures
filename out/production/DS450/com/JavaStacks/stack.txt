USES
Implement the undo feature
Build compilers(eg syntax checking)
evaluate expressions(eg 1+2*3)
build navigation(eg forward/back)

Stack works on LIFO(last in first out)
principle.

OPERATIONS

1. push(item)
   adds item on the top of the stack

2.  pop()
   removes the item on the top

3. peek()
   returns the item on the top without
   removing it from the stack

4. isEmpty()
   tells us if the stack is empty
